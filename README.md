JavaScript DOM Kullanımı

Bu repository, JavaScript DOM (Document Object Model) konusunu öğrenmek ve pratik yapmak amacıyla oluşturulmuştur.
Projede DOM üzerinde eleman oluşturma, güncelleme, event yönetimi ve local storage kullanımı gibi temel konular örneklerle gösterilmektedir.

İçerik

Bu repo aşağıdaki DOM örneklerini içerir:
	•	creating-elements.html
DOM üzerinden yeni HTML elemanları oluşturma
	•	updating-elements.html
Mevcut HTML elemanlarını güncelleme
	•	event-listeners.html
JavaScript ile event listener kullanımı
	•	filters.html
Liste elemanları üzerinde filtreleme işlemleri
	•	local-storage.html
Tarayıcı local storage kullanımı
	•	update-task.html
DOM üzerinden görev güncelleme örneği

Amaç

Bu proje aşağıdaki konuları pekiştirmek için hazırlanmıştır:
	•	DOM seçicileri kullanmak
	•	HTML elemanlarını JavaScript ile manipüle etmek
	•	Event listener kullanımı
	•	Local Storage ile veri saklama
	•	Basit etkileşimli arayüzler oluşturmak

Kullanılan Teknolojiler
	•	HTML
	•	JavaScript
	•	Browser DOM API
